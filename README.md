## 🔥 Ключевой проект: Habit Tracker

Веб-приложение для отслеживания ежедневных привычек с визуализацией прогресса и сохранением данных в браузере.

**Стек:** HTML5, CSS3 (БЭМ), JavaScript (ES6+), LocalStorage

### Основной функционал:
- Добавление/удаление привычек, отметка выполнения
- Подсчёт прогресса и фильтрация по статусу
- Сохранение данных в `localStorage`
- Модальные окна с анимациями

### Ключевые моменты реализации:
- **JavaScript:** современный синтаксис ES6+, работа с DOM и событиями (делегирование), чистые функции, замыкания, классы, работа с массивами (`map`, `filter`, `reduce`), клиентская валидация форм, `localStorage`
- **CSS:** методология БЭМ (переиспользуемые компоненты), анимации через `transition`
- **HTML:** семантическая вёрстка, доступность
